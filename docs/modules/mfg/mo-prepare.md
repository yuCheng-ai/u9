# 齐套分析与生产备料 (Material Preparation)

## 1. 概述
在制造过程中，“缺件”是导致停工待料的头号原因。齐套分析模块通过模拟计算，预先发现缺件风险；生产备料则负责将物料精准、及时地从仓库流向车间，确保生产线的高效运转。

## 2. 齐套分析 (Kitting Analysis)

### 2.1 模拟排产
- **逻辑**: 按照订单优先级模拟分配库存。
- **多维度视图**: 
  - **订单视图**: 查看哪些订单是 100% 齐套的。
  - **物料视图**: 查看哪个关键零件卡住了多少个订单。

### 2.2 预留与分配
- **硬预留**: 将库存锁定给高优先级订单，即使其他订单先开工也无法领用。
- **软分配**: 系统逻辑层面的优先占位。

## 3. 生产备料与领料流程

### 3.1 备料建议 (Material Request)
- **自动推算**: 根据 MO 需求扣除车间剩余料（线边仓库存），生成备料清单。

### 3.2 多样化领料模式
- **配套领料**: 按 MO 整套发料，适用于组装行业。
- **超领控制**: 严格按 BOM 领料，超领必须走特殊审批流程并记录损耗原因。
- **倒冲领料 (Backflush)**: 完工申报后根据 BOM 自动扣减库存，适用于螺丝、辅料等小件物料。

### 3.3 线边仓管理
- **透明化 WIP**: 物料离开总仓进入车间仓（线边仓）后，产权仍属于 MO，但物理位置变更。

## 4. 业务价值
减少生产现场的物料堆积与损耗，消除“停工待料”隐患，提升库存周转率。
