# UAP 业务开发平台 (UAP Platform)

## 1. 概述
UAP (Unified Application Platform) 是 U9 cloud 的底层架构底座。它不仅是 ERP 的运行环境，更是一个强大的低代码/无代码定制平台。UAP 使得系统能够灵活适配企业不断变化的业务流程，而无需改动核心底层代码。

## 2. 核心组件详解

### 2.1 建模工具 (Modeling Tools)
- **实体建模**: 自由定义数据库表结构，支持自动生成元数据。
- **UI 设计器**: 拖拽式界面布局，适配 PC、平板和移动端视图。

### 2.2 规则与流程引擎
- **BE 脚本 (Business Engine)**: 支持使用 C# 或可视化脚本定义复杂的业务校验逻辑。
- **工作流驱动**: 集成图形化工作流配置，支持加签、转办、条件跳转等复杂场景。

### 2.3 动态扩展能力
- **个性化扩展**: 允许用户在不影响升级的前提下，为标准单据增加自定义字段、按钮和页签。
- **多租户架构**: 确保在集团云环境下，各组织的定制逻辑互不干扰。

## 3. 数智化底座
- **API 网关**: 提供标准的 RESTful 接口，方便与第三方系统（如钉钉、企业微信、外部 WMS）集成。
- **大屏看板引擎**: 内置数据可视化组件，快速构建数字化驾驶舱。

## 4. 业务价值
降低系统 TCO（总拥有成本），缩短二次开发周期，确保 ERP 系统能够随着企业的成长而持续演进。
