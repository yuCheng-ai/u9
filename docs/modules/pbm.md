# PBM 项目制造 (Project Based Manufacturing)

PBM 模块专为按订单设计 (ETO) 和按订单制造 (MTO) 企业设计，提供从项目立项到成本核算的全生命周期管理。

## 1. 菜单结构与功能点

### 1.1 项目管理 (Project)
- **项目立项**：定义项目编码、类型、负责人及 WBS (工作分解结构)。
- **项目预算**：按 WBS 编制人工、材料、费用预算，支持多版本。
- **项目任务**：分配具体执行任务，关联进度反馈。

### 1.2 项目计划 (PPC)
- **项目 MRP**：基于项目 WBS 驱动的需求计算，确保物料专款专用（项目专用库存）。
- **项目采购/生产**：关联项目号的 PO 和 MO，支持项目间的物料借调。

### 1.3 项目成本 (PA)
- **四算对比**：概算、预算、核算、决算的多维对比分析。
- **成本归集**：按项目维度实时统计采购支出、生产投入及费用报销。

## 2. 核心业务逻辑

### 2.1 软硬关联逻辑 (Hard/Soft Pegging)
- **逻辑点**：
    - **硬关联**：物料入库时自带项目属性，非该项目 MO 不可领用。
    - **软关联**：物料可通用，但计划层面优先分配给指定项目。

### 2.2 项目库存借调逻辑
- **逻辑点**：当项目 A 急需物料而项目 B 有闲置时，支持“项目借调单”。系统自动处理项目间的成本转移与后续归还提醒。

### 2.3 按项目完工百分比 (POC) 确认收入
- **逻辑点**：对于长周期项目，支持按成本进度或里程碑确认阶段性收入与成本。
