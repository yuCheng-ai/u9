# 动态预留与可用量 (PAB)

## 概述
实时反映库存可用状态，支持精准排产。PAB (Projected Available Balance) 是库存展望的核心。

## 核心指标
- **预计可用量 (PAB)**: 考虑未来收发计划后的动态库存余额。
- **供需平衡表**: 细化的时间维度库存展望。

## 核心业务逻辑

### PAB 计算逻辑
- **公式**: `当前库存 + 预计入库 (PO/MO/调入) - 预计出库 (SO/备料/调出) = PAB`。
- **时间轴**: 系统按天、周或月展示 PAB 变化，自动识别缺料风险点。

### 动态预留 (Dynamic Reservation)
- **硬预留**: 锁定特定批次或货位，非指定单据不可挪用。
- **软预留**: 逻辑层面的分配，优先级高的单据自动抢占 ATP。
- **预留自动释放**: 当订单取消或逾期未执行时，系统自动释放预留量给其他需求。

### 库存展望与排产链接
- **逻辑点**: 生产计划 (MRP) 引擎实时读取 PAB 数据。如果某天 PAB 为负，MRP 将自动产生建议补货单（采购或生产）。
- **多组织可见性**: 销售组织可以查看生产组织的库存展望，实现真正的产销协同。
