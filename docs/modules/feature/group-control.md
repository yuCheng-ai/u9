# 集团管控与合并报表 (Group Control)

## 1. 概述
对于大型多组织企业而言，集团管控是核心诉求。U9 通过顶层的集团中心建模，实现主数据管控、内部交易抵消及财务合并。本模块旨在解决集团企业“合并报表难、内部对账难、政策下达难”的痛点，确保集团意志的一致性执行。

## 2. 核心功能详解

### 2.1 主数据管控 (Master Data Governance)
- **管控模式**: 
  - **集中分配**: 总部创建，下发给各分子公司使用。
  - **申请审核**: 分子公司申请，总部审核通过后生效。
- **范围**: 料品档案、客户/供应商、会计科目、汇率等核心基础资料。

### 2.2 内部交易与协同 (Inter-company Transactions)
- **自动对账**: 系统自动识别组织间的采购订单与销售订单，实现物流、票据和资金流的闭环对冲。
- **内部对账表**: 实时展示组织间的往来款项，自动识别不匹配项（如 A 组织已出货，B 组织未收货）。

### 2.3 财务合并 (Consolidation)
- **抵消分录自动生成**: 
  - **投资抵消**: 自动处理母公司投资收益与子公司所有者权益的抵消。
  - **往来抵消**: 自动抵消集团内部产生的应收应付、预收预付。
  - **内部销售抵消**: 自动剔除集团内部交易产生的未实现利润。
- **多准则合并**: 支持按 CAS（国内准则）和 IFRS（国际准则）分别出具合并报表。

### 2.4 资金集中管理 (Optional)
- **虚拟资金池**: 实时监控全集团各组织的银行账户余额。
- **下拨与上收**: 自动执行集团内的资金调度指令，优化资金使用效率。

## 3. 业务价值
将财务合并从“事后苦劳”转变为“过程监控”，大幅缩短结账周期，为管理层提供实时、透明的集团经营全景图。
