# 料品全生命周期管理 (Item Master)

## 1. 概述
料品是 ERP 系统中最核心的基础数据，是连接研发、采购、生产、库存、销售和财务的“灵魂”。在 U9 架构下，料品不仅仅是一个编码，它集成了复杂的业务逻辑控制属性。本模块支持从料品研发申请、属性定义、版本控制到停用归档的全生命周期管理。

## 2. 关键功能详解

### 2.1 多维度分类与档案
- **分类体系**: 支持多树状分类（管理分类、销售分类、采购分类），满足不同职能部门的统计需求。
- **全局唯一性**: 实现集团内料品编码的统一，支持“一物一码”或“多组织共享单编码”模式。

### 2.2 核心属性控制
- **业务属性定义**:
  - **供应方式**: 购买、制造、委外、虚设（Phantom）。
  - **需求来源**: MRP 计划、LRP 计划、人工订货。
  - **批次/序列号控制**: 严格控制物料的追溯精度，支持生产日期、失效日期管理。
- **库存属性**:
  - **动态预留**: 是否允许订单锁定库存。
  - **呆滞分析**: 定义呆滞天数，自动预警。

### 2.3 多单位与换算
- **主单位与业务单位**: 支持采购单位、生产单位、销售单位、计价单位等多种维度的定义。
- **换算逻辑**: 
  - **固定换算**: 如 1 箱 = 24 瓶。
  - **浮动换算**: 针对重量不固定的料品（如钢材、农产品），支持二次计量。

### 2.4 版本与工程变更
- **料品版本**: 支持 A/B/C 多版本并存，记录料品设计变更轨迹。
- **生命周期状态**: 涵盖“申请中”、“使用中”、“呆滞”、“淘汰”、“停用”等状态，控制业务环节的准入。

## 3. U9 核心竞争力：参数化配置
支持料品属性的参数化扩展，无需二次开发即可增加行业特有属性，并能参与到后续的 MRP 运算逻辑中。
