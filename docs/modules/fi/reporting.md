# 智能凭证与财务报表 (Financial Reporting)

## 1. 概述
智能凭证与报表模块是 U9“业财一体化”架构的最终落脚点。它将复杂的业务活动（如收货、发货、发票、折旧）通过**自动会计平台 (AEP)** 实时转化为标准会计分录，确保业务发生即财务感知。

## 2. 自动会计平台 (AEP - Automated Accounting)

### 2.1 凭证模板配置
- **语义化映射**: 定义业务单据的字段（如供应商分类、料品类型）如何对应到会计科目。
- **实时与异步**: 
  - **实时生成**: 业务单据审核瞬间生成凭证。
  - **批量生成**: 周期性汇总业务数据生成凭证，减轻账务压力。

### 2.2 智能对账
- **业财核对**: 自动扫描业务明细账与财务总账，高亮显示不一致项。
- **多准则转换**: 基于同一笔业务，根据 [多账簿](./multi-book) 规则自动生成 CAS 和 IFRS 两套凭证。

## 3. 多维财务报表

### 3.1 标准法定报表
- **三大表**: 资产负债表、利润表、现金流量表，支持一键出具并自动折算外币。

### 3.2 管理分析报表
- **责任中心报表**: 按部门、项目、组织维度分析盈利能力。
- **穿透式查询**: 支持从报表余额 -> 凭证 -> 原始业务单据（如一张采购入库单）的闭环溯源。

## 4. 业务价值
消除人工录入凭证带来的滞后与错误，使财务人员从繁琐的账务处理转向高价值的经营分析。
