# COST 成本管理

COST 模块提供精细化的成本核算体系，支持多成本域、多成本类型及实际成本/标准成本双轨制。

## 1. 菜单结构与功能点

### 1.1 料品成本 (IC)
- **成本域设置**：定义核算范围，关联物流组织。
- **成本项目定义**：包括材料、人工、制造费用、委外费。
- **料品成本维护**：录入标准价、计划价，支持从采购协议取价。
- **成本卷积 (Rollup)**：基于 BOM 结构，将底层零件成本逐层向上累加至成品。

### 1.2 成本会计 (CA)
- **费用分配**：将财务总账中的工资、水电费等制造费用，按分配标准（工时、产量）分摊至生产订单。
- **成本计算**：
    - 移动平均法：实时计算出库单价。
    - 月加权平均法：期末统一计算。
    - 实际成本法：按订单归集实际投入。
- **成本报表**：包括料品成本明细账、生产成本分析、毛利分析。

## 2. 核心业务逻辑

### 2.1 成本域逻辑
- **逻辑点**：成本域是财务核算的最小单位。同一成本域内的组织共享同一套料品单价。跨成本域的流动必须触发价格重估或内部交易。

### 2.2 生产成本核算流程
1. **数据准备**：确认期末所有收发货单据已过账。
2. **费用归集**：从 GL 提取制造费用。
3. **分摊规则执行**：按工位或工作中心进行费用分配。
4. **卷积计算**：处理半成品到成品的成本传递。
5. **差异分析**：对比实际成本与标准成本，分析料差、费差。

### 2.3 委外成本核算
- **逻辑点**：产成品成本 = 发出原材料价值 + 加工费发票金额。系统需自动关联委外订单与加工发票。

### 2.4 跨组织成本依赖解除
- **逻辑点**：在多工厂协同环境下，A 厂的成品是 B 厂的原料。系统需支持“一键式”多组织顺序成本计算，自动解决组织间的先后依赖关系。
