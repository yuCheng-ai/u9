# MFG 生产制造 (Manufacturing)

MFG 是面向离散制造企业的核心模块，支持多工厂协同生产、精益制造及复杂的生产计划逻辑。

## 1. 菜单结构与功能点

### 1.1 工程数据 (BOM)
- **物料清单 (BOM)**：
    - 多版本管理：支持研发 BOM (EBOM) 到制造 BOM (MBOM) 的转化。
    - 损耗率设置：支持固定损耗与比例损耗。
    - 替代料设置：支持整单替代、成组替代及优先级别。
- **工艺路线 (Routing)**：定义工序、标准工时、工作中心及资源。
- **工程变更 (ECN)**：严格控制 BOM 变更流程，支持变更生效日期管理。

### 1.2 需求计划 (MRP)
- **MRP 运算**：基于销售订单、预测、当前库存、在制、在途进行缺口计算。
- **MRP 控制台**：查看建议计划（建议生产、建议采购、建议调拨），支持一键转单。
- **批次需求计划 (LRP)**：按订单批次进行精准跟综，适用于项目型制造。

### 1.3 生产订单 (MO)
- **备料/领料**：
    - 齐套检查：下单前检查物料是否齐套。
    - 领料方式：支持倒冲 (Backflush)、领料制、送料制。
- **完工汇报**：记录生产产出、工时消耗及废品数量。
- **入库申请**：关联质量检验，判定合格后入库。

### 1.4 车间管理 (SFC)
- **工序排程**：针对关键瓶颈工序进行精细化排产。
- **工序移转**：记录产品在工序间的流转进度。
- **看板管理**：可视化展示车间任务状态、异常预警。

## 2. 核心业务逻辑

### 2.1 倒冲领料逻辑 (Backflush)
- **逻辑点**：在完工汇报时，系统根据 BOM 自动扣减线边仓库存。适用于标准化程度高、消耗稳定的物料。

### 2.2 替代料触发逻辑
- **逻辑点**：当主料缺料时，MRP 运算可自动切换至替代料，或在 MO 备料环节由人工选择替代方案。

### 2.3 生产订单重排计划
- **逻辑点**：当发生设计变更或交期调整时，系统支持 MO 的“重排计划”逻辑，自动更新所有关联工序的计划日期。

### 2.4 多产出管理 (Co-product)
- **逻辑点**：一个 MO 可以产出主产品、联产品及副产品。系统需按价值权重自动分配生产成本。
