# 组织间协同 (Collaboration)

## 概述
多组织协同是 U9 的灵魂，也是其最核心的差异化优势。在大型集团中，各个组织（如销售分公司、不同的工厂）往往是独立的核算单元，但业务上又高度耦合。U9 能够实现跨组织的业务流、物流和财务流的高度自动化。

## 1. 核心协同场景

### 1.1 内部购销协同 (Internal Buy/Sell)
- **业务流**: A 组织（销售）收到客户订单，自动向 B 组织（工厂）下达采购订单。B 组织自动生成对应的销售订单。
- **自动化**: 当 B 组织发货时，系统自动完成 A 组织的收货，并同步生成组织间的内部交易发票。
- **价值**: 消除重复录单，确保两个组织的数据实时对等。

### 1.2 直运协同 (Drop Shipping)
- **场景**: 客户向 A 组织买货，A 通知 B 组织直接把货寄给客户。
- **逻辑点**: 
  - **物流**: 从 B 仓库直接到客户。
  - **单据流**: 自动产生“销售组织 A -> 客户”的出货单，以及“供应组织 B -> 销售组织 A”的结算单。
- **结算**: 系统自动处理 A 与 B 之间的价差。

### 1.3 集中采购与委外协同
- **集中采购**: 集团总部统一采购，供应商直接发货给各成员工厂，总部负责结算。
- **跨组织委外**: A 组织将某道关键工序发给 B 组织加工，B 组织在系统内被视为一个特殊的“内部供应商”。

## 2. 组织间结算规则 (Settlement Rules)
- **结算价策略**: 
  - **固定结算价**: 预设的内部协议价。
  - **成本加成**: 在供应组织实际成本的基础上加上固定毛利。
- **自动对账**: 系统实时记录组织间的债权债务，月底一键生成对账单。

## 3. 组织间物流可见性
- **在途库存**: 销售组织可以实时查询生产组织的在制品 (WIP) 和成品库存展望 (PAB)，实现精准的交付承诺。
- **库存借调**: 支持跨组织紧急库存调拨。

## 4. 财务自动并账与抵销
- **逻辑点**: 在合并报表层面，系统能够自动识别所有携带“内部交易号”的凭证。
- **抵销**: 自动抵销内部往来（应收/应付）和内部交易损益，生成集团真实的资产负债表和利润表。

## 5. 核心竞争力：真正的业财一体化
与通过外部集成实现的协同不同，U9 的协同是底层的单据级关联。这意味着每一笔跨组织业务在发生的那一刻，物流、业务流和财务流就已经在系统内完成了闭环。
