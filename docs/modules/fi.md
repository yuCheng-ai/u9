# FI 财务管理 (Financials)

FI 模块提供多会计准则、多账簿、多币种的全球化财务核算体系，支持自动会计处理。

## 1. 菜单结构与功能点

### 1.1 总账管理 (GL)
- **凭证处理**：手工录入、模板生单及业务自动生单。
- **科目体系**：支持多套科目表，支持辅助核算项（部门、人员、项目、客户、供应商）。
- **多账簿并行**：同时维护实体账簿、税务账簿与管理账簿。
- **期末结转**：汇兑损益核算、损益结转、自动对账。

### 1.2 应收管理 (AR)
- **销售发票**：关联销售出库单生成，支持红字发票、拆分/合并开票。
- **收款核销**：支持预收款管理、应收款冲销、多币种核销差额处理。
- **账龄分析**：实时监控应收风险。

### 1.3 应付管理 (AP)
- **采购发票**：关联采购收货单生成，支持三单匹配（PO、收货单、发票）。
- **付款申请**：支持多级审批流。
- **供应商对账**：定期生成对账单，核对往来余额。

### 1.4 固定资产 (FA)
- **资产卡片**：记录原值、折旧方法、使用部门、存放地点。
- **折旧计算**：支持平均年限法、双倍余额递减法等。
- **资产变动**：处理资产调拨、清理、减值准备。

### 1.5 全球合并 (GC)
- **合并范围定义**：定义母子公司持股关系。
- **抵销分录**：自动识别并抵销内部往来、内部交易毛利。
- **合并报表**：生成集团合并资产负债表、利润表。

## 2. 核心业务逻辑

### 2.1 自动会计平台 (AEP)
- **逻辑点**：业务单据（如收货、出货）不直接写凭证，而是抛送到“自动会计平台”。平台根据配置的“入账规则”异步生成会计凭证，确保业务与财务的解耦。

### 2.2 跨组织核销
- **场景**：客户在 A 组织欠款，在 B 组织付款。
- **逻辑点**：系统支持跨组织收款核销，自动生成组织间的内部往来凭证（其他应收/其他应付）。

### 2.3 实时汇兑损益
- **逻辑点**：外币业务单据发生时，系统实时记录本币金额。期末按月末汇率统一调整，自动生成汇兑损益凭证。

### 2.4 财务合并抵销
- **逻辑点**：基于“内部交易号”追踪跨组织业务，在合并时自动匹配并冲销双方的债权债务及损益。
