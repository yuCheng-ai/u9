# 工序汇报与移动点管理 (Shop Floor Reporting)

## 1. 概述
工序汇报是制造执行系统（MES）在 ERP 中的核心延伸，负责采集生产现场的“黑盒”数据。通过汇报，管理层可以实时监控每个工单、每个工作中心、每个工序的进度、质量和效率。

## 2. 关键功能详解

### 2.1 工序汇报 (Operation Reporting)
- **内容采集**:
  - **产出量**: 合格数、报废数、返修数。
  - **工时采集**: 准备工时、运行工时（支持手动录入或与设备集成采集）。
  - **人员信息**: 记录谁在什么时间执行了该任务。
- **触发逻辑**: 支持逐道工序汇报，或关键工序（里程碑）汇报。

### 2.2 移动点管理 (Move Logic)
- **工序流转**: 控制物料从工序 10 移动到工序 20 的准入条件。
- **移动类型**:
  - **自动移动**: 汇报后自动流转到下一道工序。
  - **检验移动**: 必须经过工序质检后方可向下流转。

### 2.3 现场在制品 (WIP) 监控
- **堆积分析**: 实时发现哪道工序积压了大量半成品，定位生产瓶颈。
- **报废处理**: 汇报时录入报废原因，自动触发成本核销逻辑。

## 3. 数智化集成 (IoT/MES)
U9 支持与扫描枪、平板电脑或 PLC 设备集成，实现“无纸化报工”，极大提升现场数据的实时性与准确性。

## 4. 业务价值
实现生产进度的全透明管理，为精细化成本核算提供准确的工时数据基础。
