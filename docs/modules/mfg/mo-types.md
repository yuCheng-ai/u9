# 生产订单管理 (MO Management)

## 1. 概述
生产订单（Manufacturing Order）是制造执行的核心指令。它不仅承载了“生产什么、生产多少、何时完工”的任务，更集成了物料领用清单、工序流转计划和成本归集路径。U9 支持从简单组装到复杂多级加工的全场景制造控制。

## 2. 生产场景与订单类型

### 2.1 标准生产订单
- **逻辑**: 基于标准 BOM 和工艺路线生成的任务。
- **应用**: 大规模重复性制造，追求标准成本控制。

### 2.2 返工订单 (Rework Order)
- **逻辑**: 针对检验不合格或客户退回的产品进行修复。
- **控制**: 支持手动添加返工料件，不计入标准产出统计。

### 2.3 改制订单 (Modification)
- **逻辑**: 将一种现有半成品或成品通过改制转换为另一种产品。
- **价值**: 快速响应客户个性化需求，减少旧版库存积压。

## 3. 核心功能详解

### 3.1 订单生命周期控制
- **开立/审核**: 校验物料有效性与工艺合规性。
- **下达 (Release)**: 触发库存占用，生成齐套分析任务。
- **完工/结案**: 清理现场在制品（WIP），将订单成本结转至库存成本。

### 3.2 动态备料与缺料分析
- **齐套分析**: 开工前扫描全阶物料，根据缺料情况调整排产顺序。
- **替代逻辑**: 允许在 MO 层面临时指定替代料，解决突发性短缺。

### 3.3 成本实时归集
- **料工费透明**: 每一笔领料、每一次工序报工都实时计入该订单的 WIP 账户，实现“日结”甚至“时结”的成本监控。

## 4. 业务价值
通过对生产订单的精细化管控，企业能够实时掌握生产现场的瓶颈点，确保交付计划的准确性。
