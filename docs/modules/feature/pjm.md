# 项目制造 (PJM)

## 概述
以项目为主线的全生命周期管理，支持“四算”体系。

## PJM 四算
- **概算**: 项目启动前的初步预算。
- **预算**: 详细的项目执行预算。
- **核算**: 实际发生的项目成本。
- **决算**: 项目结束后的盈利分析。

## 核心业务逻辑

### 软硬关联逻辑 (Hard/Soft Pegging)
- **硬关联**: 物料入库时自带项目属性，非该项目 MO 不可领用。
- **软关联**: 物料可通用，但计划层面优先分配给指定项目。

### 项目库存借调逻辑
- **逻辑点**: 当项目 A 急需物料而项目 B 有闲置时，支持“项目借调单”。系统自动处理项目间的成本转移与后续归还提醒。

### 按项目完工百分比 (POC) 确认收入
- **逻辑点**: 对于长周期项目，支持按成本进度或里程碑确认阶段性收入与成本。
